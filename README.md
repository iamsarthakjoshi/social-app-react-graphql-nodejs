# Simple Social App based on React, Graphql, Node, Typescript and more.

> This project is for learning purpose only.

### Features

- Login
- Register
- Post Message
- Post Comment
- Update Message
- Delete Message

# Client Side

### Tech Stack

- React
- Graphql Client
- Typescript Language

# Server Side

### Tech Stack

- Node
- Graphql Server (includes express by default)
- ORM (TBD)
- MongoDB
